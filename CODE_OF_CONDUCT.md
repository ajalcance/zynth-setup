# Code of Conduct

We pledge to make participation in `zynth-setup` a harassment-free experience for everyone.

We expect: empathy and kindness, respect for differing viewpoints, graceful acceptance of
constructive feedback, and a focus on what is best for the community. We do not tolerate
harassment, insulting/derogatory comments, or publishing others' private information.

Report unacceptable behavior to the maintainers. This Code of Conduct is adapted from the
[Contributor Covenant](https://www.contributor-covenant.org), version 2.1.
